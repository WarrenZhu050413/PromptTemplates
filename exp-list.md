List all expansions (snippets and macros) in the system.

Run this command:
```bash
python ~/.claude/expansions-dev/expansions_cli.py list --output json
```

This will show:
- All expansion IDs
- Names and descriptions
- Trigger patterns
- Whether they're enabled
- Priority levels
- Output modes

**Execute the list command now.**
